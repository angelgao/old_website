Windows Azure POC

An web application with single sign-on authentication, deployed using Windows Azure.

This was another proof-of-concept project created for Metrolinx to access the possibilities of using Windows Azure as an authentication service for future projects.
The web application consists of a sign-in feature that can redirect users to use other identity providers such as Hotmail or Facebook which then maps to an organization-specific ID. In addition, I experimented with various services on Windows Azure, such as web application deployment and databases.


The project was successful in familiarizing my supervisor with the technology and helping him make a decision regarding the services to use for upcoming Metrolinx web applications.

Content Delivery Network

A proof of concept project that mimics a content delivery network made for Metrolinx testing.

A web solution of MCV web applications developed with ASP.net. It consists of 3 layers: a WCF (Windows Communication Foundation) web service, a DMZ layer and an edge layer client. Information is communicated using SignalR hubs so that it can be displayed in the edge layer in real-time, and new input is stored in databases through the WCF service. I have also implemented a feature to dynamically change the communication protocol (basic HTTP, WS HTTP, Net.Tcp) used by the WCF service.


I performed various time tests to determine the efficiency of the combination of these technologies. The data collected will help in making decisions regarding which technologies to use in future Metrolinx web applications.
